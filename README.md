#  MedVoice AI — Smart Medical Audio Transcription Platform

> *Transforming doctor–patient conversations into precise, structured medical insights using AI.*

---

##  Overview

**MedVoice AI** is an intelligent web-based application that converts medical audio conversations into accurate, readable text using advanced speech recognition.

Designed for healthcare scenarios, this system enhances documentation, improves efficiency, and ensures clarity in medical communication.

---

##  Key Features

 **Audio-to-Text Transcription**
Convert medical audio recordings into text using AI-powered speech recognition.

 **Download Transcripts**
Export transcriptions instantly as text files.

 **Audio Preview Player**
Listen to uploaded audio directly within the application.

 **User-Based History Tracking**
Securely stores previous transcriptions for each user.

 **Authentication System**
Login-based access ensures personalized and secure usage.

---

##  User Interface

✔ Modern split-screen dashboard
✔ Left panel: Audio upload & controls
✔ Right panel: Live transcription output
✔ Clean, responsive, and user-friendly design

---

##  Tech Stack

**Frontend**

* HTML5
* CSS3 (Modern UI Styling)
* JavaScript (Dynamic Interaction)

**Backend**

* Python (Flask)

**AI Model**

* OpenAI Whisper (Speech-to-Text)

**Database**

* SQLite (User & History Management)

---

## ⚙️ How to Run Locally

```bash
# Clone the repository
git clone https://github.com/yourusername/medical-voice-translator.git

# Navigate to project folder
cd medical-voice-translator

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
